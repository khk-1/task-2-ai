# Color Recognition using OpenCV

## Overview
This project demonstrates real-time color recognition using OpenCV and Python. The application captures live video from the webcam and detects three colors: Red, Green, and Blue. When a color is detected, a bounding box and the color name are displayed on the screen.

## Features
- Real-time webcam color detection
- Detects Red, Green, and Blue colors
- Displays bounding boxes around detected objects
- Shows the detected color name
- Built using OpenCV and NumPy

## Technologies Used
- Python 3
- OpenCV
- NumPy
- Anaconda (Virtual Environment)
- Visual Studio Code

## Installation

1. Create and activate your Anaconda environment.
2. Install the required packages:

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
