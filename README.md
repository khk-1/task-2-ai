# Color Recognition using OpenCV

## Overview
This project demonstrates real-time color recognition using OpenCV and Python. The application captures live video from the webcam and detects Red, Green, and Blue colors. When a color is detected, a bounding box and the color name are displayed on the screen.

## Features
- Real-time webcam color detection
- Detects Red, Green, and Blue colors
- Displays bounding boxes around detected objects
- Shows the detected color name

## Technologies Used
- Python
- OpenCV
- NumPy
- Anaconda
- Visual Studio Code

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
